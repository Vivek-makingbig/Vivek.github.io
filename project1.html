Fire-Sensing Alarm with Arduino

One of my recent projects involved creating a Fire-Sensing Alarm using Arduino. This project demonstrates my ability to work with sensors and develop functional safety systems. The alarm detects the presence of fire or smoke and triggers an alert, making it a practical and useful tool for fire safety.

Key Features:

  • Fire/Smoke Detection: Utilized a flame sensor to detect fire or smoke in the environment.
  • Alert Mechanism: An audible alarm is triggered when a fire is detected.
  • Arduino-based Implementation: Developed using an Arduino board with efficient code.
  • Real-time Monitoring: The sensor continuously monitors for fire hazards.

Components Used:

  • Arduino Uno
  • Flame sensor
  • Buzzer (for the alarm)
  • LEDs (for visual indication)
  • Resistors, jumper wires, and breadboard

This project taught me valuable lessons in integrating hardware with software to build practical solutions. It also enhanced my understanding of sensor-based systems and microcontroller programming.
